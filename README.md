# ☁️ CloudBuddy – AI Learning Buddy for Cloud Computing Basics

## 📖 Project Overview

CloudBuddy is an AI-powered Learning Buddy developed as part of the **Infosys Springboard AI EMPOW(H)ER Capstone Project**. It is designed to help beginners understand the fundamentals of Cloud Computing through simple explanations, real-life examples, and interactive quizzes.

The application provides an engaging and user-friendly learning experience by simplifying technical concepts and encouraging self-paced learning.

---

## 🎯 Objectives

The primary objectives of this project are:

- Explain Cloud Computing concepts in simple language.
- Help beginners learn through interactive conversations.
- Provide real-life examples to improve understanding.
- Assess learners using quizzes.
- Give instant feedback on quiz performance.
- Encourage self-learning and continuous improvement.

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/your-username/CloudBuddy-AI-Learning-Buddy.git
```

### Move into the project folder

```bash
cd CloudBuddy-AI-Learning-Buddy
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
streamlit run app.py
```

The application will open automatically in your browser.

---

## 🚀 Deployment

This project can be deployed using **Streamlit Community Cloud**.

### Deployment Steps

1. Push the project files to GitHub.
2. Open Streamlit Community Cloud.
3. Sign in using GitHub.
4. Click **New App**.
5. Select the repository.
6. Choose **app.py** as the main file.
7. Click **Deploy**.
8. Copy the generated public URL.
